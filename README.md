# git-ref

Get the current git reference via [git describe](https://git-scm.com/docs/git-describe).

## Installation

```sh
yarn add git-ref
```

## Usage

```js
import { gitRef } from 'git-ref';

const ref = gitRef();

console.log(ref);
```
